
# Automated Customer Segmentation — 25 Campaign Optimization Version

This repository contains a bachelor thesis demo pipeline for automated customer segmentation from natural language campaign descriptions.

## Setup and run

1. Fill OPENAI_API_KEY in run_pipeline.py file.
2. Fill CAMPAIGN_PROMPT string variable with the desired campaign to test in run_pipeline.py file.
3. Optionally fill CAMPAIGN_ID field to identify campaign in the output folder after the results come through.
4. ```bash run.sh ```
