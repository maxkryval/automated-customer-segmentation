
# Automated Customer Segmentation

This repository contains a bachelor's thesis demo pipeline for automated customer segmentation from natural language campaign descriptions.

## Setup and run
1. Clone the repository.
2. Fill ```OPENAI_API_KEY``` in run_pipeline.py file.
3. Fill ```CAMPAIGN_PROMPT``` string variable with the desired campaign to test in ```run_pipeline.py``` file.
4. Optionally fill ````CAMPAIGN_ID``` field to identify the campaign in the output folder after the results come through.
5. In terminal - ```bash run.sh```
